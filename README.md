# Project Title

Accodal Technical Assessment simple webpage

## Getting Started

### Dependencies

- Node v18.18.2
- NPM v8.13.2

### Executing program

- Clone Repo

```
git clone https://github.com/davecar21/accodal-webpage.git
```

- Install package

```
npm i
```

- Run script

```
npm start
```

- Open your browser on [http://localhost:4200/](http://localhost:4200/)
